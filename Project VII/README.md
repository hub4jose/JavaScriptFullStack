# A Static Node.js and Express Site

This website host 5 mini projects created with Node Js ,JavaScript , PUG and Express

## Getting Started
npm start on the console and localhost:3000 on the browser


### Prerequisites
JavaScript , html , css

## Technologies used
JavaScript
PUG
Express
CSS
